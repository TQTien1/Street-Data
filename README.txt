Hướng dẫn sử dụng phần mềm:
- Mở TrafficCluster.exe
- Import file Street Data.csv
- Chọn ô tròn trong sidebar để chọn biểu đồ mức độ giao thông
- Kéo thả thanh ở mục Parameters để update bản đồ
- Bấm "Find" để tìm cặp tham số tốt nhất
- Điền tham số vào form và bấm submit
- Bấm create map để tạo bản đồ
- Trên bản đồ bấm chọn các ô tròn để hiển thị bản đồ mức độ giao thông
